# Minimal DAO on Ethereum

Minimal Ethereum DAO using scaffold-eth and Tailwind CSS

## Getting started

```bash
# install dependencies
$ yarn

# 1. Terminal
# start local chain
$ yarn chain

# 2. Terminal
# deploy contracts
$ yarn deploy

# start app
$ yarn start
```
