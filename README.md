# Minimal NFT DAO on Ethereum

Minimal Ethereum DAO using an ERC721 NFT as membership + vote.
Built with scaffold-eth and Tailwind CSS.

## Getting started

```bash
# install dependencies
$ yarn

# 1. Terminal
# start local chain
$ yarn chain

# 2. Terminal
# deploy contracts
$ yarn deploy

# start app
$ yarn start
```
